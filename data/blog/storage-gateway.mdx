>AWS Storage Gateway proporciona un conjunto de características que te permiten aprovechar el almacenamiento de AWS de manera efectiva en tus aplicaciones y flujos de trabajo existentes. Proporciona un conjunto estándar de protocolos como ISCSI y NFS, que te permiten usar tus aplicaciones existentes sin realizar cambios.

- A través de su caché local, Gateway proporciona acceso de baja latencia a los datos que se usan con frecuencia.

- El Gateway optimiza la transferencia de datos al almacenamiento de AWS, como la optimización de la transferencia a través del almacenamiento en búfer inteligente, la administración de carga para adaptarse a las variaciones de la red y la administración del ancho de banda.

- Gateway proporciona un mecanismo efectivo para almacenar datos en AWS a lo largo de la gama de servicios de almacenamiento más adecuados para tus casos de uso.

- Gateway es fácil de implementar y puede usar tu infraestructura virtual existente e inversiones en hipervisores, o puede instalarse en tu centro de datos o en oficinas remotas como un dispositivo de hardware.

- El software de Gateway se ejecuta como una máquina virtual o en el dispositivo de hardware sin estado, lo que te permite crear y administrar fácilmente nuevas instancias de tu gateway a medida que evolucionan tus necesidades de almacenamiento.

- Finalmente, el servicio está integrado de forma nativa en los servicios de administración de AWS como Amazon Cloudwatch, AWS Cloudtrail, AWS Key Management Service (KMS) y AWS Identity and Access Management (IAM).
